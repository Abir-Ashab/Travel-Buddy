# Travel Platform API Reference

**Base URL:** `http://localhost:3000/api`  
**Auth:** Bearer token required (except public endpoints)  
**Roles:** SUPER_ADMIN, ADMIN, TRAVELER, EXPLORER

## Authentication (`/api/auth`)
- `POST /register` - Register new user (Public)
- `POST /login` - User login (Public)  
- `GET /refresh-token` - Refresh JWT token (Public)

## Users (`/api/users`)
- `POST /create-admin` - Create admin user (ADMIN+)
- `DELETE /account` - Delete user account (ADMIN+)
- `PUT /upgrade` - Upgrade to traveler (ADMIN+, EXPLORER)
- `PUT /status` - Update user status (ADMIN+)
- `PUT /profile` - Update user profile (All)
- `GET /profile` - Get user profile (All)
- `DELETE /profile` - Delete user profile (ADMIN+)
- `GET /reports` - Get post reports (ADMIN+)
- `PUT /reports/:reportId` - Resolve report (ADMIN+)
- `DELETE /reports/:reportId` - Delete reported post (ADMIN+)

## Posts (`/api/posts`)
- `GET /` - Get all posts with filters (All)
- `GET /featured` - Get featured posts (All)
- `GET /:id` - Get post by ID (All)
- `GET /:id/details` - Get post with details (All)
- `POST /` - Create new post (ADMIN+, TRAVELER)
- `PUT /:id` - Update post (ADMIN+, TRAVELER)
- `DELETE /:id` - Delete post (ADMIN+, TRAVELER)
- `POST /:id/like` - Like post (All)
- `DELETE /:id/like` - Unlike post (All)
- `POST /:id/save` - Save post (All)
- `DELETE /:id/save` - Unsave post (All)
- `POST /:id/share` - Share post (All)
- `GET /user/my-posts` - Get user's posts (All)
- `GET /user/liked-posts` - Get user's liked posts (All)
- `GET /user/saved-posts` - Get user's saved posts (All)
- `POST /:id/report` - Report post (ADMIN+, TRAVELER)
- `PATCH /:id/feature` - Toggle feature status (ADMIN+, TRAVELER)
